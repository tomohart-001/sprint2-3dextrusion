Site Inspector Tools Guide:

  1. Site Boundary: Draw or load your property boundary to begin analysis. Use "Draw Site Boundary" to create new polygons or "Load Existing Boundary" to retrieve saved data
  2. Property Setbacks: Select front and back edges of your site boundary, then enter setback requirements to calculate buildable area
  3. Structure Placement: Define the building footprint by drawing its intended location within the site boundary. Create 3D building visualizations with adjustable height settings
  4. Map Controls: Switch between map styles (Streets, Satellite, Outdoors, Light) and toggle 3D buildings with adjustable opacity
  5. Measure Tool (📐): Click two points on the map to measure distances between them
  6. Site Information (ℹ️): View detailed site data including council, zoning, area, slope, and aspect information
  7. Panel Controls: Use the gear icon (⚙️) to open/close the inspector panel with all analysis tools

Workflow: Start by drawing or loading a site boundary, then define setbacks to calculate buildable area. Draw floor plans to visualize your building design.
